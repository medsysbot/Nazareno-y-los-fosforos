# Nazareno y los fósforos

Minijuego educativo orientado a la toma de decisiones seguras frente al uso del fuego.

## Propósito

Este proyecto NO busca incentivar el uso de fósforos.

Su objetivo es:

- Identificar visualmente entornos peligrosos
- Entender qué materiales pueden provocar incendios
- Entrenar la toma de decisiones rápidas
- Fomentar la reacción inmediata ante un error (apagar el fuego o pedir ayuda)

---

## Mecánica

El jugador prende un fósforo y debe decidir dónde tirarlo.

### Zonas seguras
- Agua
- Cemento

### Zonas peligrosas
- Hojas secas (alto riesgo)
- Leña (riesgo medio)
- Fence de madera (riesgo bajo)

Si se comete un error:
- el juego permite una **segunda oportunidad**
- el jugador puede reaccionar y apagar el fuego
